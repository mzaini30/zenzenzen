Pada Maret 2023, IndiHome berhasil meraih berbagai penghargaan Public Relation Indonesia Award! Nggak tanggung-tanggung hingga 10 gelar berhasil disabet oleh IndiHome! Penghargaan apa ajakah itu?

Menurut [web review gadget](https://www.pricebook.co.id/article/news/13060/10-gelar-pria-award-2023-buktikan-indihome-terbaik-dan-inovatif), terdapat 9 penghargaan yang berhasil diraih oleh Public Relation IndiHome:

Lima kategori di antaranya adalah:

- PR-Corporate PR Program melalui program 3C (Connections, Creativity, Charity)
- PR-Digital PR Program melalui Just on IndiHome (JOIN)
- PR Marketing Program PR Blog Competition 2022

Kemudian, IndiHome juga meraih juara pada:

- Saluran Aplikasi Digital
- Manajemen krisis dan pasca krisis
- Profil video media khusus

Lalu, pada pengembangan berbasis masyarakat CSR, ada 3 program inovatif IndiHome yang telah diakui:

- Limitless Esport Academy (LEAD)
- IndiHome Gideon Badminton Academy
- IndiHome Friends
