Mungkin banyak yang mengira bahwa mengenkripsi file itu sulit. Sebenarnya nggak sih 😅. Eh tapi sebelumnya, apa sih enkripsi file itu?

Kalau menurut ChatGPT:

Enkripsi file adalah proses mengubah informasi dalam suatu file menjadi bentuk yang tidak dapat dibaca oleh pihak yang tidak berwenang 🔒. Tujuan dari enkripsi file adalah untuk melindungi informasi yang sensitif dan rahasia, sehingga hanya orang yang memiliki kunci enkripsi yang dapat membacanya 🔑. Dalam enkripsi file, informasi dalam file asli diubah menggunakan suatu algoritma enkripsi yang kompleks menjadi bentuk yang tidak dapat dimengerti 🤔. Orang yang ingin membaca file tersebut harus memiliki kunci enkripsi yang tepat untuk dapat membaca file yang sudah dienkripsi tersebut 🔐.

Intinya sih, file yang kita miliki ini disamarkan. Jangan sampai bisa dilihat sama orang lain isinya. Itu 😊.

## Langsung ke Teknis

👉🏻 Pertama, install dulu Node JS dan pnpm. Untuk install Node JS, bisa langsung ketik di Google dengan kata kunci: Download Node JS. Lalu, ketika sudah menginstall Node JS, install pnpm dengan perintah:

```bash
npm i -g pnpm
```

Jangan lupa juga, install Git Bash 😊. Karena, aku udah nyoba aplikasi ini menggunakan Git Bash. Nggak tau kalau pakai Command Prompt bisa apa nggak. Untuk menginstallnya, monggo cari di Google: Install Git Bash 🧑‍💻.

Sekarang, kita masuk ke folder yang berisi file yang ingin kita kunci 🔐. Misalnya aja, di folder tersebut ada file gambar-rahasia.jpg 🤫. Nah, itu yang mau kita enkripsi. Jadi, gimana caranya?

Pertama, buat file kunci.txt yang berisi dengan passwordnya 🔑.

Lalu, buka Git Bash dan ketikkan ini: 💻

```bash
pnpm kunci gambar-rahasia.jpg
```

Jadi, maksud kode tersebut adalah memasukkan file gambar-rahasia.jpg ke dalam list file yang akan dikunci. 📁🔒

Untuk menguncinya, tinggal jalankan: 👇

```bash
pnpm kunci
```

Kalau untuk membuka, tinggal jalankan:

```bash
pnpm buka
```

Oh iya, intinya kan di file kunci.txt. Itu adalah kuncinya. Jadi, setelah mengunci file, jangan lupa dihapus ya file kunci.txt nya. 🤫🔒🔑🚫
