## foto profil

untuk foto profil, klik edit pada gadget "foto profil". lalu, upload foto profil yang diinginkan.

![](/gambar-untuk-postingan/foto-profil-gadget.png)

## postingan

untuk postingan, pada "bagian judul", gunakan titik koma untuk memisahkan judul dengan deskripsi. contoh: "rumah mewah; rumah bagus tengah kota"

kemudian, untuk menuliskan link google mapsnya, tuliskan linknya pada "bagian label"

![](/gambar-untuk-postingan/isinya.png)

## nomor wa

untuk memasukkan nomor wa, klik edit pada gadget "nomor wa". kemudian, masukkan nomor wa-nya. contoh: 628123456789

![](/gambar-untuk-postingan/nomor-wa.png)

## slide

untuk memasukkan slide, klik edit pada gadget "slide". kemudian, tambah link dengan format seperti berikut:

- gambar slide = linknya
- caption slide = judul link

![](/gambar-untuk-postingan/link-slide.png)

## link lainnya

untuk memasukkan link lainnya, klik edit pada gadget "link lainnya". kemudian, tambah link dengan format seperti berikut:

- pada judul link -> judul; deskripsi

![](/gambar-untuk-postingan/link-link.png)
